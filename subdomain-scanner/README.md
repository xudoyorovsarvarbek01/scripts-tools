# Subdomain Scanner

A lightweight subdomain enumeration tool built for educational and authorized security testing purposes.

This project is part of my cybersecurity learning journey, focused on reconnaissance automation and domain analysis.

---

## Features

- Subdomain discovery
- Domain format validation
- Host availability checking
- Simple CLI usage
- Lightweight and fast

---

## 🛠 Tech Stack

- Bash
- Linux environment (Kali / Ubuntu)
- Standard networking utilities

---

## Purpose

This tool was created to:

- Practice reconnaissance techniques
- Automate basic enumeration tasks
- Understand how attackers discover assets
- Improve scripting skills for penetration testing

---

## Usage

```bash
./subscanner.sh example.com /path/to/wordlist.txt
```
