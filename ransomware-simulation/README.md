# 🛡 Ransomware Simulation (Python)

A Python-based educational simulation demonstrating how ransomware-style encryption can be implemented.

This project exists strictly for cybersecurity research and learning purposes.

---

## Objective

To understand:

- How ransomware encrypts user files
- How cryptographic libraries are used in Python
- How attackers automate file targeting
- How defenders can detect and prevent such behavior

---

## Concepts Covered

- File traversal
- Encryption techniques
- Key handling logic
- Attack workflow simulation

---

## Defensive Value

By building and analyzing this simulation, I aim to:

- Better understand real-world ransomware attacks
- Learn how to detect suspicious file activity
- Improve incident response knowledge
- Strengthen malware analysis skills

## Overview

This simulation:

- Encrypts `.txt` and `.pdf` files
- Demonstrates file traversal
- Shows how encryption logic is automated
- Includes a decryption module

It helps security students understand how ransomware operates from a technical perspective.

---

## Requirements

- Python 3.x
- Linux / Windows (tested in lab VM)
- Isolated test directory

Check Python version:

```bash
python3 --version
```

## Usage

**Encrypt Files**
```python
python3 encryptor.py
```
**Decrypt Files**
